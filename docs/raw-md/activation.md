# Запуск и выход

{% include notitle [neuroexpert-button](_includes/reusables/neuroexpert-button-99270fdba323.md) %}


Когда пользователь запускает навык, Алиса передает слово навыку до тех пор, пока он не будет деактивирован.

## Как запустить навык {#activate}

Разговор с навыком всегда начинает пользователь — навык только отвечает на запросы. Есть два способа начать разговор:

- Просто запустить — пользователь просит Алису запустить навык, навык запускается и приветствует пользователя.
- Запустить с командой — пользователь сразу говорит, в каком навыке и что надо сделать.

Навык активируется в ответ на фразы:

{% list tabs %}

- Просто запустить

    - «запусти навык [активационное имя]»;
    - «запусти чат с [активационное имя]»;
    - «давай поиграем в [активационное имя]»;
    - «сыграем в [активационное имя]».

    Пример: «Алиса, поиграем в города».

- Запустить с командой

    - «скажи [активационное имя] [команда]»;
    - «узнай у [активационное имя] [команда]»;
    - «попроси [активационное имя] [команда]»;
    - «спроси у [активационное имя] [команда]».

    Пример: «Спроси Сбербанк, где ближайшее отделение».

    Вы получите команду в запросе от пользователя в свойстве [request.command](https://yandex.ru/dev/dialogs/alice/doc/ru/request-simpleutterance.md#request-desc). При активации навыка в это свойство попадает весь текст, кроме активационной фразы. Если пользователь просто скажет `спроси Сбербанк` — свойство `request.command` в запросе будет пустым.

{% endlist %}

Все команды смотрите в [консоли](https://dialogs.yandex.ru/developer/). Откройте навык → вкладка **Настройки** → раздел **Публикация в каталоге** → поле **Примеры запросов**.

[Навыки умного дома](https://yandex.ru/dev/dialogs/smart-home/doc/ru/) не требуют активационной фразы: они работают как встроенные сценарии Алисы. Например, «Алиса, включи свет на кухне».

## Учтите при настройке навыка {#dont-forget}

- Все возможные активационные имена. Например, для названия S7 — «эс сэвэн», «эс семь» и «авиакомпания сибирь». Так Алиса быстрее научится распознавать название навыка.
- Активационные имена в правильных склонениях. Например, для навыка «Крокодил» можно добавить вариант «крокодил**а**» и использовать его во фразе «давай поиграем в крокодила».
- В некоторых приложениях, например в [приложении Яндекс](https://yandex.ru/promo/searchapp/promopage/desktop?rdr=mobile), для каждого нового навыка Алиса открывает отдельную вкладку.

## Как выйти из навыка {#deactivate}

Когда разговор подошел к концу, передайте в ответе флаг `"end_session": true`.

Пользователь в любой момент может попросить Алису выйти из навыка фразами «Алиса, вернись», «Алиса, хватит», «Стоп», «Закончить» или «Выйти».

Диалог прекратится автоматически, если навык:

- Прислал ответ, который не соответствует [формату](https://yandex.ru/dev/dialogs/alice/doc/ru/response.md).
- Не успел ответить за 4,5 секунды.


## Часто задаваемые вопросы {#troubleshooting}

{% cut "Можно ли запустить несколько навыков сразу?" %}

Пока открыт один навык Алисы, недоступны как другие навыки, так и сама Алиса. Чтобы разговор закончился, навык нужно [деактивировать](https://yandex.ru/dev/dialogs/alice/doc/ru/activation.md#deactivate).

{% endcut %}

{% cut "Что делать, если Алиса не находит навык по активационному имени?" %}

Если Алиса не находит навык по активационному имени, проверьте:

- распознает ли она ваше активационное имя;
- правильно ли указаны склонения активационного имени;
- соблюдены ли все [требования к активационному имени](https://yandex.ru/dev/dialogs/alice/doc/ru/requirements.md#key-phrase).

{% endcut %}

{% cut "Можно ли протестировать навык голосом до публикации?" %}

Да. Для этого сделайте навык [приватным](https://yandex.ru/dev/dialogs/alice/doc/ru/test.md#test-private-skill). Проверьте его работу на поверхностях, которые вы указали в настройках навыка и на которых вы авторизовались.

Протестировать, как Алиса распознает [активационное имя](https://yandex.ru/dev/dialogs/alice/doc/ru/activation.md), можно во время обычного разговора. Публиковать навык необязательно.

Чтобы опубликовать навык в каталоге, измените тип доступа на **Публичный**, проверьте навык на соответствие [формальным требованиям](https://yandex.ru/dev/dialogs/alice/doc/ru/requirements.md) и отправьте его на модерацию.

{% endcut %}
