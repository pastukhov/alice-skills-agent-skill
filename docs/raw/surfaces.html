    Где работают навыки

    Вашим навыком будут пользоваться на разных поверхностях. Это устройства (например, Яндекс Станция, головное устройство автомобиля) и сервисы Яндекса (например, Яндекс Браузер, Яндекс Навигатор).

    У приложений и устройств разные возможности:

    Тип поверхности

    Что может навык

    Пример устройства или приложения

    С экраном

    Показывать изображения и отправлять ссылки:

    {
    "meta": {
        "interfaces": {
            "screen": { }
        }
}

    Внимание

    Если навык работает только на устройствах с экраном, в настройках публикации выберите опцию Нужно устройство с экраном.

    Алиса на Windows, приложение Яндекс, Яндекс Браузер

    Без экрана

    Общаться только голосом:

    {
    "meta": {
        "interfaces": { },
}

    Если вы хотите показывать изображение пользователям с экраном, отправьте корректно заполненное свойство response.text для тех, кто изображение увидеть не сможет.

    Яндекс Навигатор, Яндекс Станция, Яндекс Авто

    При поиске навыков в каталоге пользователь может отфильтровать их в зависимости от устройства. Чтобы оставить лучшее впечатление о навыке, приспосабливайте его к различным устройствам.

    Пример диалога с Алисой

    Ваш навык — игра, где нужно выбрать верный ответ на вопрос. Для поверхностей с экраном предлагайте варианты с изображениями, а без экрана — короткие вопросы, которые легко воспринимать на слух.

    Устройство с экраном

    Пользователь: Алиса, давай поиграем в игру Угадай город.

    Алиса: Привет! Назови свое имя.

    Пользователь: Иван.

    Алиса: Приятно познакомиться, Иван. Отгадай, что это за город:

    Пользователь: Казань.

    Алиса: Верно! Сыграем еще?

    Устройство без экрана

    Пользователь: Алиса, запусти навык Загадки.

    Алиса: Замечательно! Я знаю много разных загадок, но, если вы захотите загадать мне что-нибудь сами — просто скажите «Алиса, отгадай мою загадку». Чур я первая: Под гору — коняшка, в гору — деревяшка.

    Пользователь: Санки.

    Алиса: Абсолютно точно! Продолжаем?
