# Проект публичного навыка Алисы для кофейни

## Концепция

Навык помогает гостям кофейни узнать меню, адреса, часы работы и оставить заявку на обратный звонок. Рабочее название: `Кофейня Зерно рядом`. Если используется реальный бренд, нужно подтвердить права через сайт для верификации.

Основные сценарии:

- узнать меню;
- найти адрес;
- узнать часы работы;
- оставить заявку на обратный звонок;
- получить помощь;
- выйти из навыка.

## Активационные фразы

Рекомендуемые варианты:

- `Кофейня Зерно рядом`
- `Кофейня Зерно`
- `Зерно рядом`
- `Кафе Зерно`

Примеры запуска:

- `Алиса, запусти навык Кофейня Зерно рядом`
- `Алиса, спроси у Кофейня Зерно, что есть в меню`
- `Алиса, узнай у Кофейня Зерно адреса`
- `Алиса, попроси Кофейня Зерно перезвонить мне`

## Интенты

| Интент | Примеры | Поведение |
| --- | --- | --- |
| `MENU` | `меню`, `что есть из кофе`, `покажи напитки` | Рассказывает категории меню |
| `ADDRESSES` | `адреса`, `где вы находитесь` | Перечисляет адреса |
| `HOURS` | `часы работы`, `до скольки открыты` | Называет график |
| `CALLBACK_START` | `перезвоните мне`, `оставить заявку` | Объясняет сбор данных и начинает заявку |
| `HELP` | `помощь`, `что ты умеешь` | Объясняет возможности |
| `EXIT` | `стоп`, `выйти`, `закончить` | Завершает сессию |

## Сценарий диалога

### Первый запуск

Пользователь: `Алиса, запусти навык Кофейня Зерно рядом`

Алиса: `Здравствуйте! Это навык кофейни Зерно. Я могу рассказать меню, адреса, часы работы или принять заявку на обратный звонок. Что хотите узнать?`

Саджесты: `Меню`, `Адреса`, `Часы работы`, `Обратный звонок`.

### Меню

Пользователь: `Что есть в меню?`

Алиса: `В меню есть эспрессо, американо, капучино, латте, раф, фильтр-кофе, чай, лимонады, круассаны, сырники и десерты. Хотите узнать напитки, еду или сезонные предложения?`

Если есть экран, можно показать карточку или кнопки, но голосовой ответ должен быть полноценным.

### Адреса

Пользователь: `Где вы находитесь?`

Алиса: `У нас три кофейни: на Тверской, 12, на Покровке, 8 и на Ленинском проспекте, 45. Какая кофейня вам ближе?`

### Часы работы

Пользователь: `До скольки вы работаете?`

Алиса: `Кофейни на Тверской и Покровке работают каждый день с 8:00 до 22:00. Кофейня на Ленинском проспекте работает с 9:00 до 21:00. Хотите узнать адреса?`

### Обратный звонок

Сценарий явно предупреждает, какие данные собираются и зачем.

Пользователь: `Оставить заявку на звонок`

Алиса: `Хорошо. Я передам сотруднику кофейни ваше имя и телефон, чтобы он перезвонил по заявке. Как вас зовут?`

Пользователь: `Анна`

Алиса: `Анна, продиктуйте номер телефона.`

Пользователь: `Восемь девятьсот девятнадцать сто двадцать три сорок пять шестьдесят семь`

Алиса: `Проверьте, пожалуйста: Анна, номер 8 919 123 45 67. Все верно?`

Пользователь: `Да`

Алиса: `Спасибо, заявка принята. Сотрудник кофейни перезвонит вам в рабочее время.`

### Fallback

Пользователь: `А у вас уютно?`

Алиса: `Я пока могу помочь с меню, адресами, часами работы и заявкой на обратный звонок. Что подсказать?`

После повторной ошибки: `Попробуйте сказать: меню, адреса, часы работы или обратный звонок.`

### Выход

Пользователь: `Стоп`

Алиса: `Хорошо, до встречи в кофейне Зерно!`

Webhook должен вернуть `"end_session": true`.

## Состояние диалога

```json
{
  "last_intent": "MENU",
  "callback": {
    "step": "awaiting_phone",
    "name": "Анна",
    "phone": null,
    "selected_branch": "Покровка, 8"
  },
  "last_response_key": "menu_categories"
}
```

## Требования к модерации

- Название навыка: 2-5 слов, кроме подтвержденного бренда из одного слова.
- Дополнительные активационные имена: 2-4 слова.
- Если используется бренд кофейни, нужен сайт для верификации прав.
- Описание должно быть на русском языке и точно соответствовать функциям навыка.
- В описании нужен контакт для обращений по работе навыка.
- Команды `Помощь` и `Что ты умеешь` должны работать.
- Функциональность должна совпадать с описанием.
- Ответы должны быть полноценными без экрана.
- Кнопки, саджесты и ссылки должны вести туда, куда обещают.
- На запрос `ping` нужно быстро отвечать без внешних зависимостей.

Нельзя:

- писать `лучший кофе в городе`, `самая вкусная кофейня`, `номер один`, если это субъективно или не подтверждено;
- использовать чужой бренд, логотип или фото без прав;
- собирать телефон без объяснения цели;
- делать переход на сайт единственной целью навыка.

## Тестовый чеклист

- [ ] Навык запускается по основному и дополнительным активационным именам.
- [ ] При пустой команде после запуска навык показывает приветствие.
- [ ] Пользователь получает меню, адреса и часы работы.
- [ ] Меню не слишком длинное для голосового ответа.
- [ ] Пользователь может оставить заявку на обратный звонок.
- [ ] Навык объясняет, зачем собирает имя и телефон.
- [ ] Навык просит подтвердить номер.
- [ ] Команды `Помощь`, `Что ты умеешь`, `Повтори` работают.
- [ ] Непонятная фраза возвращает пользователя к доступным действиям.
- [ ] Команды `стоп`, `выйти`, `закончить` завершают сессию.
- [ ] Навык понятен на устройстве без экрана.
- [ ] Кнопки имеют рабочий `payload` или корректную ссылку.
- [ ] Описание на русском языке и совпадает с функциональностью.
- [ ] Нет неподтвержденных заявлений вроде `лучший`, `самый`, `номер один`.
- [ ] Каждый ответ содержит `response.text`, `response.end_session` и `"version": "1.0"`.
