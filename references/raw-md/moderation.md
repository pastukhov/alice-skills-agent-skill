# Модерация

{% include notitle [neuroexpert-button](_includes/reusables/neuroexpert-button-99270fdba323.md) %}


Проверка навыка занимает до трех дней. Ее результат будет отображен в [консоли разработчика](https://dialogs.yandex.ru/developer/). Навыки, которые находятся на модерации, нельзя редактировать, но вы можете в любой момент вернуть навык в статус черновика: откройте страницу навыка и нажмите кнопку **Вернуть в разработку**.

Если у вас возникли вопросы к модерации, [напишите нам](https://yandex.ru/dev/dialogs/alice/doc/ru/feedback.md). Выберите тему «Процесс модерации диалога Алисы».

## Что проверяют {#moderation-checks}

1. Описание навыка соответствует всем [формальным требованиям](https://yandex.ru/dev/dialogs/alice/doc/ru/requirements.md).
    
1. Поддержаны [команды «Помощь» и «Что ты умеешь»](https://yandex.ru/dev/dialogs/alice/doc/ru/requirements.md#content).
    
1. Функциональность навыка соответствует его описанию и инструкциям помощи.
    
1. Если навык публикуется в каталоге и в блоке **Ограничения на доступные платформы** не выбрана опция **Нужно устройство с экраном**, ответы навыка полноценны и понятны на устройствах без экрана с чатом (на Станции, в Навигаторе и т. п.).
1. Для навыков с авторизацией — как работает сценарий авторизации. Перед тем как отправить навык на модерацию, заполните поле **Заметки для модератора**. Подробно опишите, по какому сценарию будет выполняться авторизация в навыке. Укажите авторизационные данные (логин и пароль) для проверки сценария. Пример, как заполнить поле **Заметки для модератора**
    ```
    Для запроса авторизации нужно сказать «Запросить баланс».
    Логин: test@test.ru.
    Пароль: password.
    После успешной авторизации навык должен сообщить баланс.
    Навык может общаться с пользователем и без авторизации — рассказывать про тарифы
    или сообщать адрес отделений.
    ```
    

## Модерация брендированных навыков {#brand}

Если вы создаете навык для организации или торговой марки, вы можете использовать названия брендов, их логотипы и прочие атрибуты. При модерации такого навыка проверят, соответствует ли он дополнительным требованиям:

- В названии навыка, в активационных именах, в описании и на иконке используется один и тот же бренд.
- Если при поиске по имени бренда его официальный сайт попадает в первые три позиции, он указан в поле **Сайт для верификации прав использования бренда**.
- Права на сайт, указанный как **Сайт для верификации прав использования бренда**, [подтверждены](https://yandex.ru/support/webmaster/service/rights.html) в Яндекс Вебмастере.


## Проверка после модерации {#check-after-moderation}

После публикации навыка Яндекс Диалоги периодически проверяют, что навык работает и корректно отвечает на запросы. Если навык не отвечает несколько часов, его отключат.

Для проверки платформа присылает запрос со словом `ping` в свойстве `original_utterance`:

```json
{
  "request": {
    "command": "",
    "original_utterance": "ping",
    "type": "SimpleUtterance"
  }
}
```

Свойство `text` в ответе вашего навыка может содержать любую строку (например, `ok`, `1` или `pong`) — по формальному ответу платформа поймет, что навык исправен.

Советуем вынести обработку запроса `ping` в отдельную [ветку сценария](https://yandex.ru/dev/dialogs/alice/doc/ru/write-scenario.md#extra-branch), где навык не будет проверять другие команды, соединяться с базами данных и подключать сторонние библиотеки, а сразу завершит работу. Это ускорит ответ навыка, снизит нагрузку на сервер и позволит избежать затрат при платных запросах в базы данных.

## Часто задаваемые вопросы {#troubleshooting}

{% cut "Почему навык отключили после публикации?" %} 

После публикации навыка Яндекс Диалоги периодически проверяют его работу. Если навык не отвечает несколько часов, его отключат. Проверьте его на соответствие [протоколу работы](https://yandex.ru/dev/dialogs/alice/doc/ru/protocol.md), исправьте ошибки и повторно отправьте на [модерацию](https://yandex.ru/dev/dialogs/alice/doc/ru/moderation.md).

{% endcut %}

{% cut "Почему навык не прошел модерацию?" %} 

Навык не пройдет модерацию, если его настройки не соответствуют [требованиям](https://yandex.ru/dev/dialogs/alice/doc/ru/moderation.md). Чтобы увидеть причину, выберите навык в [консоли разработчика](https://dialogs.yandex.ru/developer/) и перейдите на вкладку **Общие сведения** → **Анализ результатов**.

{% endcut %}

{% cut "Как отправить приватный навык на модерацию?" %} 

Чтобы отправить приватный навык на автомодерацию, выберите навык в [консоли разработчика](https://dialogs.yandex.ru/developer/) и нажмите **Опубликовать**.

{% endcut %}

{% cut "Для чего нужны команды «Помощь» и «Что ты умеешь»?" %} 

Это универсальные команды. Они сориентируют пользователя и расскажут об основных возможностях навыка. Это позволяет удержать пользователя в навыке и помочь ему вернуться в основную ветку сценария после ошибки голосового ввода. Подробнее о командах см. в разделе [Содержание навыка](https://yandex.ru/dev/dialogs/alice/doc/ru/requirements.md#content).

{% endcut %}
